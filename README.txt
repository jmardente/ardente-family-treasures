ARDENTE FAMILY TREASURES BOUTIQUE PACKAGE

FILES INCLUDED
- shop.html
- assets/cora-and-the-coral-reef-cover.png

UPLOAD TO GITHUB
1. Download and unzip this package.
2. Open: https://github.com/jmardente/ardente-family-treasures
3. Click Add file, then Upload files.
4. Drag shop.html into the upload box.
5. Drag the assets folder into the upload box too.
6. Scroll down and click Commit changes.

CONNECT STRIPE
1. Open shop.html in GitHub.
2. Click the pencil icon to edit.
3. Find these placeholders:
   REPLACE-BOOK-LINK
   REPLACE-SIGNED-LINK
   REPLACE-BUNDLE-LINK
4. Replace each full placeholder address with its matching live Stripe Payment Link.
5. Update the sample prices to match Stripe.
6. Click Commit changes.

IMPORTANT
Never paste a Stripe secret API key into shop.html.
Use only Stripe Payment Links that begin with https://buy.stripe.com/
