HOW TO UPLOAD TO GITHUB ON A MAC
1. Double-click the ZIP file to unzip it.
2. Open the unzipped folder.
3. Open your GitHub repository in Safari or Chrome.
4. Click Add file > Upload files.
5. Drag EVERYTHING INSIDE the unzipped folder into GitHub.
6. Click Commit changes.
7. Wait 1–5 minutes and refresh the GitHub Pages website.

IMPORTANT
- Upload the individual files and assets folder, not only the ZIP.
- Keep index.html in the main/root area of the repository.
- Keep the folder name exactly: assets
- The current secure purchase link goes to Amazon.
- The browser cart works for browsing, but the site does not collect card payments.
